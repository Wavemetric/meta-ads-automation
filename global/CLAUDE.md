# Wavemetric AX팀 Claude Code 가이드

## 팀 소개
Wavemetric AX팀. 커머스 종합 대행(마케팅/운영) 및 자사 브랜드 이커머스.
팀원 대부분 비개발자 출신. 업무 자동화와 데이터 분석 위주로 Claude Code 활용.

---

## 기본 플랫폼

아래 플랫폼을 기본으로 사용한다. 다른 선택이 필요한 경우 먼저 팀에 공유한다.

| 목적 | 플랫폼 |
|------|--------|
| 버전 관리 | GitHub (Private repo) |
| 웹 배포 | Vercel |
| 데이터베이스 | Supabase |
| 배치/자동화 | GitHub Actions |
| 데이터 분석 스크립트 | Python (로컬 또는 GitHub Actions) |

## 반드시 지키는 규칙

### 비용 발생 전 사전 협의
Claude API, OpenAI API, Apify 등 외부 API 사용 비용이 발생하는 구조로
개발이 필요한 경우, 구현 전 담당자와 먼저 확인한다.

### 보안 사고 즉시 알림
API 키 유출 등 보안 사고가 발생하면 숨기지 않고 즉시 담당자에게 알린다.
빠르게 알릴수록 피해가 줄어든다.

### 서버 이관 기준
로컬에서 정상 동작 확인 후, 주 1회 이상 반복 실행이 필요한 스크립트는
서버 이관 요청. 그 이하는 로컬 자동 실행으로 충분하다.
